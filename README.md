Estructura del Proyecto
La solución se organiza de la siguiente manera:
EjercicioA.cs (Sección A): Contiene respuestas teóricas y reflexiones sobre conceptos básicos, documentadas dentro de comentarios en el código.
EjercicioB.cs (Sección B): Análisis de flujo de ejecución y seguimiento de variables en bloques de código predefinidos.
EjercicioC.cs (Sección C): Resolución de problemas de sintaxis y lógica. Incluye fragmentos de código donde se identificaron y corrigieron errores.
EjercicioD.cs (Sección D): Implementación de lógica faltante en métodos y estructuras de control incompletas.
EjercicioE.cs (Sección E): Desarrollo de una solución práctica desde cero basada en los requerimientos finales de la guía.
Program.cs: Clase principal que orquestal la ejecución de los ejercicios.
